Quick Summary (after first setup)
Every time you want to run the project, just open two terminals:

Terminal 1 — Backend:
cd backend
uvicorn app.main:app --reload --port 8000

Terminal 2 — Frontend:
npm run dev


If you want to run tests or lint/format code during development:
cd backend
pip install -r requirements-dev.txt

For just running the project (production/demo), you only need:
pip install -r requirements.txt
